Korby Probe


The Korby Probe is a Selfcrafted original 3d printer add-on that provides a low-cost Z-Probe solution to any 3d printer on the market. It is also compatible with the roadkill toolhead

Checkout Korby on Github

The probe is fairly simple to put together
What's in the package:
  • 4x M2.5 Screws
  • 4x M2.5 Nuts
  • 4x M3 Countersunk 10mm Magnets N35
  • 1x Kailh Sword GM X Micro Switch 100M Switch pre-soldered
  • 1x JST-XH 2 pin female header pre-soldered


As visible in the picture, 2 diagonally opposite holes are left empty for the sake of mounting the probe onto your toolhead.
Currently the probe is pre-compatible with the following toolheads:
  • Ratrig V-Core 3 (EVA3) - on Printables by Animatrix
  • More coming soon, if you've added compatibility please let us know and we'll add it here
